
List of scripts:
(1) basickernel.m - A simple script which illustrates the truncated kernel density estimator
(2) truncatedRenyiestimate.m - Renyi entropy estimator using truncated kernel density estimate
->(3) truncatedShannonestimate.m - Shannon entropy estimator using truncated kernel density estimate
(4) knn_Renyi_entropy_estimate - Script to illustrate existing Renyi entropy estimators including entropic graph estimator, Leonenko's estimator
->(5) calcualteweightgeneral.m - Script to calculate optimal weights of the ensemble
->*(6) msecompare_PD_parallel_knn.m - Script to calculate variation of MSE vs sample size 
->*(7) msecompare_PD_parallel_knn_varyingd.m - Script to calculate variation of MSE vs dimension d
(8) aucloop.m - Script to calculate area under ROC curves (see figure 3 in the paper)
(9) perfcurve.m, areaundercruve.m - Utility functions used in aucloop.m
